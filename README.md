# 🚀 FE to AI Engineer - 我的 AI 工程师成长之路

这个仓库记录了我从 **1年前端开发** 转型为 **AI 应用/全栈工程师** 的完整学习路径、日常笔记及实战项目。

## 🎯 学习目标

- 掌握大模型（LLM）底层调用、函数调用与结构化输出。
- 理解并落地 RAG（检索增强生成）与向量检索技术。
- 熟练运用 LangChain、LangGraph 构建有状态的复杂 Multi-Agent 系统。
- 结合前端优势，打通大模型流式传输（Streaming）与极佳的用户交互（UX）。
