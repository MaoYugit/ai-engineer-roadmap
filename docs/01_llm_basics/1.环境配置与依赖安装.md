### 第一步：配置环境变量与安装依赖

我们将同时配置 Python 和 Node.js 的开发环境。因为 DeepSeek 的 API 与 OpenAI 完全兼容，所以我们将使用 `openai` SDK 来调用它。

---

#### 1. 根目录配置 `.env`

在你的项目**最外层根目录**（即 `01_llm_basics` 的上一级目录）下，创建一个名为 `.env` 的文件，写入你的 DeepSeek 密钥与接口地址：

```env
DEEPSEEK_API_KEY="你的_deepseek_api_key_在此"
DEEPSEEK_BASE_URL="https://api.deepseek.com"
```

---

#### 2. Python 环境准备

打开终端，进入 `python` 目录，创建虚拟环境并安装 `openai`（调用大模型）与 `python-dotenv`（读取环境变量）库：

```bash
cd 01_llm_basics/python
py -m venv .venv

# 激活虚拟环境 (macOS/Linux)
source .venv/bin/activate
# 激活虚拟环境 (Windows)
# .venv\Scripts\activate

pip install openai python-dotenv
```

---

#### 3. Node.js 环境准备

打开另一个终端，进入 `nodejs` 目录，初始化项目并安装 `openai`、`dotenv`（读取环境变量）以及 `tsx`（直接运行 TS 脚本的工具）：

```bash
cd 01_llm_basics/nodejs
pnpm init -y
pnpm add openai dotenv
pnpm add -D typescript tsx @types/node
```

并在 `01_llm_basics/nodejs` 目录下新建一个 `tsconfig.json` 文件，写入最基础的 TS 配置：

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "esModuleInterop": true,
    "strict": true,
    "skipLibCheck": true
  }
}
```
